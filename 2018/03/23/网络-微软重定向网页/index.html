#网络# 微软重定向网页

发表于 2018-03-23 更新于 2020-10-27 分类于 Web

http://www.msftconnecttest.com/redirect

这个网址可以快速从定向到校园网等需要验证网络的登陆界面，虽然设备都会弹出提示，但是如果错过提示就会导致无法再打开登陆界面（某些比较细心的网站访问任何网页都能重定向到登陆界面，但是校园网不是 括弧笑） 该地址提取自Windows跳转页，请放心使用。
