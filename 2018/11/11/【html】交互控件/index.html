②表单

表单是一种要求用户输入的可视化控件集合

创建一个表单

创建表单需要引入form元素，通过更改form元素的属性可以初步确定数据的发送对象、发送方式等信息。

form

[Dou.Blo]

常用属性：

属性

说明

action

该属性确定发送数据的目标页面地址

method

该属性确定发送数据的方式，有get和post可选，默认为post前者为数据一次性传送，后者为分步传送。但get保密性差，且有数据限制，所以推荐使用post

novalidate

指定是否为用户输入该form中的控件内容进行检查

input

[Sig.Blo] 需要注意的是，input元素需要嵌套在form元素中才可用！
