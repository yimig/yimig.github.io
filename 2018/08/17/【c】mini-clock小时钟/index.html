【C#】Mini clock小时钟

发表于 2018-08-17 更新于 2020-10-27 分类于 Project.IDEA ， %e5%89%8d%e5%a0%82 ， 从零开始的代码学习计划

该程序可以创建一个小时钟窗口，可以使您实时关注时间，支持调整字体颜色、大小、位置等设置，特别适合将状态栏隐藏的用户使用。   程序名：小时钟（Mini Clock） 体积：96KB（不包含后期生成的数据文件） 兼容性：Microsoft .NET Framework 4.5 以上（Windows 7，Windows Server 2008，Windows Server 2008 R2，Windows Vista，Windows 8/8.1/10) 支持的系统中可能需要手动安装Microsoft .NET Framework 4.5，请参阅官方声明

调整： ⚪修复解锁位置再锁定位置后鼠标穿透失效的bug ⚪修复应用不完全隐藏的bug 下载：小时钟

目前实现的功能：

⚪利用母窗口调整时钟的位置 ⚪调整时钟的透明度、大小 ⚪调整时钟的颜色 ⚪将母窗口隐藏在状态栏，通过下划窗口实现。
