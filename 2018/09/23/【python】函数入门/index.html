函数，对应C#中的方法，是在日常编程工作中最广泛使用的工具，如何巧妙地分割问题形成函数是一个程序员必不可少的素质，今天来学习Python的函数构成，了解其与C#等语言的差异，可为日后编程工作带来极大方便。
